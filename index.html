Read this before using the tool

This calculator is an academic demonstration built from a Master's thesis (Determinants of Under-Five Mortality in Bangladesh and its Predictions: Using Machine Learning Models) and is intended for educational use only.

It does not replace medical or public health advice and must not be used to make decisions about an individual child's health or care. If you have concerns about a child's health, please consult a qualified healthcare provider.
